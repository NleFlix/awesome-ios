## Installation 

### Requirements 

- Swift 4.2
- iOS 9.0 

### Cocoapods

ElegantUI is available through [CocoaPods](http://cocoapods.org). To install it, simply add the following line to your Podfile:

```ruby
platform :ios, '9.0'
pod "ElegantUI"
```
