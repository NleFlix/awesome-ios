## Installation 

### Requirements 

- Swift 4.2 ( Swift 4.2 -> use 'version 1.0.0' )
- iOS 9.0 

### Cocoapods

GGUI is available through [CocoaPods](http://cocoapods.org). To install it, simply add the following line to your Podfile:

```ruby
platform :ios, '9.0'
pod "GGUI"
```
